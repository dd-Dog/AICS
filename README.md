# 智能计算系统课程实验

## 项目简介

本项目为智能计算系统课程实验练习工作空间，包含实验代码、数据、报告和文档。

## 目录结构

```
AICS/
├── src/              # 源代码目录
│   ├── algorithms/   # 算法实现
│   ├── utils/        # 工具函数
│   └── models/       # 模型定义
├── data/             # 数据文件目录
│   ├── raw/          # 原始数据
│   ├── processed/    # 处理后数据
│   └── results/      # 结果数据
├── reports/          # 实验报告目录
│   └── lab_reports/  # 各实验报告
├── docs/             # 文档目录
│   ├── requirements/ # 需求文档
│   └── design/       # 设计文档
└── experiments/      # 实验脚本目录
    └── notebooks/    # Jupyter notebooks
```

## 使用说明

1. 将实验代码放在 `src/` 目录下
2. 将实验数据放在 `data/` 目录下
3. 将实验报告放在 `reports/` 目录下
4. 使用 `experiments/` 目录存放实验脚本和notebooks

## 提交说明

- 实验报告：提交到 `reports/lab_reports/` 目录
- 代码文件：提交到 `src/` 目录
- 数据文件：如需要，提交到 `data/` 目录
