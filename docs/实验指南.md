# 智能计算系统课程实验指南

## 实验前准备

### 1. 环境配置

#### Python 环境
- 推荐使用 Python 3.8 或更高版本
- 建议使用虚拟环境（venv 或 conda）

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

#### Jupyter Notebook（可选）
如果使用 Jupyter Notebook 进行实验：

```bash
pip install jupyter notebook
jupyter notebook
```

### 2. 项目结构理解

熟悉项目的目录结构，了解各个目录的用途：
- `src/`: 存放源代码
- `data/`: 存放数据文件
- `reports/`: 存放实验报告
- `experiments/`: 存放实验脚本

## 实验流程

### 1. 创建实验分支（如使用Git）

```bash
git checkout -b lab01
```

### 2. 编写实验代码

- 在 `src/` 目录下创建相应的模块
- 在 `experiments/` 目录下创建实验脚本
- 遵循代码规范，添加必要的注释

### 3. 准备实验数据

- 将原始数据放在 `data/raw/` 目录
- 处理后的数据放在 `data/processed/` 目录
- 实验结果放在 `data/results/` 目录

### 4. 运行实验

```bash
# 运行实验脚本
python experiments/run_experiment.py

# 或使用 Jupyter Notebook
jupyter notebook experiments/notebooks/lab01.ipynb
```

### 5. 撰写实验报告

- 使用 `reports/实验报告模板.md` 作为模板
- 将完成的报告放在 `reports/lab_reports/` 目录
- 报告应包含：
  - 实验目的和原理
  - 实验步骤和代码
  - 实验结果和分析
  - 实验总结

## 常见问题

### Q: 如何安装特定版本的包？
A: 在 `requirements.txt` 中指定版本号，然后运行 `pip install -r requirements.txt`

### Q: 数据文件太大，无法提交到Git？
A: 大文件已在 `.gitignore` 中排除，可以手动管理或使用 Git LFS

### Q: 如何运行 Jupyter Notebook？
A: 确保已安装 jupyter，然后在项目根目录运行 `jupyter notebook`

### Q: 实验报告格式要求？
A: 使用提供的模板，包含所有必需章节，代码需要添加注释

## 实验提交

### 提交内容
1. 源代码（`src/` 目录）
2. 实验报告（`reports/lab_reports/` 目录）
3. 实验脚本（`experiments/` 目录）
4. 必要的配置文件

### 提交格式
- 实验报告：PDF 或 Markdown 格式
- 代码文件：Python 源文件（.py）
- 数据文件：根据要求决定是否提交
